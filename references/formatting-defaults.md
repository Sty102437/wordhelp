# Formatting Defaults

Applied when the user does not specify preferences. The agent asks before using these.

---

## Chinese Academic (默认)

Based on course-paper-builder standards. All Chinese: 宋体; all Western/numbers: Times New Roman.

### Body text
| Property | Value |
|----------|-------|
| CN font | 宋体 (SimSun), 12pt (小四) |
| EN font | Times New Roman, 12pt |
| Line spacing | 1.5x |
| First-line indent | 2 Chinese characters (24pt) |
| Alignment | justify (两端对齐) |

### Headings
| Level | Font | Size | Bold | Alignment | Built-in Style |
|-------|------|------|------|-----------|----------------|
| Heading 1 (章节) | 宋体 | 16pt (三号) | yes | justify | `Heading 1` |
| Heading 2 | 宋体 | 15pt (小三) | yes | justify | `Heading 2` |
| Heading 3 | 宋体 | 14pt (四号) | no | justify | `Heading 3` |

### Abstract / 摘要
| Property | Value |
|----------|-------|
| Title font | 宋体, 16pt (三号), bold |
| Title alignment | center (居中) |
| Title style | Do NOT use Heading style (avoids auto-numbering in TOC) |
| Body font | 宋体, 12pt, justify, 1.5x line spacing |
| Keywords | `关键词：` bold, content regular 12pt |

### Abstract (English)
| Property | Value |
|----------|-------|
| Title font | Times New Roman, 16pt, bold, center |
| Body font | Times New Roman, 12pt, justify |
| Keywords | `Keywords:` bold, content regular |

### Figures and Tables
| Property | Value |
|----------|-------|
| Table caption (above) | 宋体, 10.5pt (五号), bold, center ("表1-1 数据对比") |
| Figure caption (below) | 宋体, 10.5pt (五号), center ("图1-3 系统架构") |
| Equation number | right-aligned via tab stop, e.g. (1-1), (2-3) |
| Equation | centered |

### References
| Property | Value |
|----------|-------|
| Title | 宋体, 16pt, bold, center (no Heading style) |
| Items | 宋体, 10.5pt (五号), justify |
| Format | `[N] Author. Title[J]. Journal, Year, Vol(Issue): Pages.` |

### Page layout
| Property | Value |
|----------|-------|
| Page size | A4 |
| Margins | top/bottom 2.54cm, left 3.17cm, right 3.17cm |
| Page break | Abstract on new page after cover/scoring |
| Page break | References on new page after body |
| Page numbers | None on cover + scoring page. Body starts from 1. |
